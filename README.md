## 1. 서비스 개요

**암호 생성기**는 사용자가 원하는 조건에 맞는 안전한 비밀번호를 즉시 생성하고,  
생성된 비밀번호를 사이트 정보와 함께 로컬에 저장·조회할 수 있는 **클라이언트 전용 웹 애플리케이션**입니다.

| 항목 | 내용 |
|------|------|
| 플랫폼 | Web (Desktop / Mobile 반응형) |
| 언어 | 한국어 |
| 저장 방식 | 브라우저 `localStorage` (서버 없음) |
| 인증 방식 | 마스터 비밀번호 (로컬 단순 검증) |

## 2. 주요 기능

### 2-1. 암호 생성
- 암호 **길이** 지정 (최소 4자 ~ 최대 50자, 기본값 12자)
- 포함 문자 유형 선택 (체크박스):
  - ☑ 대문자 (A–Z)
  - ☑ 소문자 (a–z)
  - ☑ 숫자 (0–9)
  - ☑ 특수문자
- 특수문자 **개별 토글** 선택: `!` `@` `#` `$` `%` `&` `*`
- 특수문자 **직접 입력** (추가 커스텀)
- 생성된 암호 **클립보드 복사** (복사 후 2초간 피드백 표시)

### 2-2. 비밀번호 저장
- 사이트명과 비밀번호를 묶어 로컬에 저장
- 암호 생성 후 비밀번호 입력란에 **자동 연동**
- 저장 성공/실패 시 **인라인 피드백 메시지** 표시 (2초 후 자동 소멸)

### 2-3. 저장된 목록 조회 (보관함)
- **마스터 비밀번호** 입력 후 잠금 해제
  - 최초 접근 시: `prompt()` 로 마스터 비밀번호 신규 설정
  - 이후 접근 시: 입력값 검증 (일치 여부 확인)
- 저장된 목록을 테이블 형태로 표시 (사이트 / 비밀번호 / 삭제)
- 행별 **삭제** 기능 (confirm 없이 즉시 삭제)
- 저장 항목 없을 시 **Empty State** 화면 표시

## 3. 화면 구성

```
┌─────────────────────────────────┐
│  [ICON]  암호 생성기             │  ← Header
│          안전한 비밀번호를 생성...│
├─────────────────────────────────┤
│  ⚙️ 생성 옵션                   │
│                                 │
│  길이: [  12  ]  (4~50자)       │  ← LengthInput
│                                 │
│  ☑ 대문자  ☑ 소문자             │  ← CheckboxGroup
│  ☑ 숫자    ☑ 특수문자           │
│                                 │
│  특수문자 선택                  │
│  [!][@][#][$][%][&][*]          │  ← SymbolToggle (활성=흑)
│                                 │
│  직접 입력: [예: !@#          ] │  ← CustomSymbolInput
│                                 │
│  [   생성하기   ]  [ 복사 ]     │  ← GenerateButton / CopyButton
│                                 │
│  ┌ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┐  │
│    생성된 암호: Xk9@mL#2pQ     │  ← ResultDisplay
│  └ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┘  │
└─────────────────────────────────┘
```

```
┌─────────────────────────────────┐
│  💾 비밀번호 저장                │
│                                 │
│  사이트:    [google.com       ] │  ← text input
```

## 4. 사용자 흐름 (User Flow)

### 4-1. 암호 생성 흐름

```
시작
 │
 ├─ 길이 설정 (기본 12)
 ├─ 문자 유형 선택 (체크박스)
 ├─ 특수문자 선택 (토글 or 직접 입력)
 │
 ▼
[생성하기 클릭]
 │
 ├─ 선택 조건 기반 암호 생성 (랜덤 셔플)
 ├─ ResultDisplay에 표시
 └─ SaveForm 비밀번호란 자동 입력
```

### 4-2. 저장 흐름

```
[저장하기 클릭]
 │
 ├─ 사이트 or 비밀번호 비어있음? ──→ 경고 메시지 표시 (2초)
 │
 └─ 값 있음
     ├─ localStorage["vault"] 에 { site, pw } 추가
     └─ "저장 완료!" 메시지 표시 (2초)
```

### 4-3. 보관함 접근 흐름

```
[확인 클릭]
 │
 ├─ localStorage["master"] 없음?
 │   └─ prompt()로 신규 설정 → localStorage 저장
 │
 └─ master 있음
     ├─ 입력값 일치 → Vault 목록 표시 (페이드인)
     └─ 불일치 → "비밀번호가 틀렸습니다" 인라인 표시
```

## 5. 컴포넌트 목록

| # | 컴포넌트명 | 유형 | 설명 |
|---|-----------|------|------|
| 1 | `Header` | Layout | 앱 아이콘 + 타이틀 + 서브텍스트 |
| 2 | `LengthInput` | Input | number type, min=4 / max=50, 기본값 12 |
| 3 | `CheckboxGroup` | Control | 대문자·소문자·숫자·특수문자 4종 체크박스 (기본 전체 ON) |
| 4 | `SymbolToggle` | Control | 특수문자 개별 토글 버튼 7개 (활성=흑, 비활성=백) |
| 5 | `CustomSymbolInput` | Input | 추가 특수문자 자유 입력 (text type) |
| 6 | `GenerateButton` | Action | Primary CTA — 조건 기반 암호 생성 |
| 7 | `CopyButton` | Action | Secondary — 클립보드 복사, 2초간 "✓ 복사됨" 표시 |
| 8 | `ResultDisplay` | Display | mono 폰트 결과 박스 / 생성 전 Empty State 포함 |
| 9 | `SaveForm` | Form | 사이트 + 비밀번호 입력 + 저장 버튼 |
| 10 | `InlineFeedback` | Feedback | 성공(녹색) / 오류(적색) 인라인 메시지, 2초 자동 소멸 |
| 11 | `MasterPasswordInput` | Input | password type, Enter 키 제출 지원 |
| 12 | `VaultTable` | Display | 3컬럼 테이블 (사이트·비밀번호·삭제) + Empty State |
| 13 | `DeleteButton` | Action | 행별 삭제, confirm 없이 즉시 처리 |

## 6. 데이터 구조

### localStorage — `"vault"`

저장된 비밀번호 목록을 JSON 배열로 관리합니다.

```json
[
  { "site": "google.com",  "pw": "Xk9@mL#2pQ" },
  { "site": "naver.com",   "pw": "P$w3rD!92z" }
]
```

### localStorage — `"master"`

보관함 잠금에 사용하는 마스터 비밀번호 (평문 문자열).

```
"myMasterPassword123"

// 미설정 시: null (최초 접근 시 prompt()로 설정)
```

> ⚠️ **보안 주의:** 현재 마스터 비밀번호는 평문으로 저장됩니다.  
> 실서비스 적용 시 bcrypt 등의 해싱 처리가 필요합니다.

## 8. 제약 사항 및 유의점

| 구분 | 내용 |
|------|------|
| 🔒 보안 | 비밀번호 및 마스터 PW가 `localStorage`에 **평문** 저장됨. 민감 정보 보관에 부적합 |
| 🌐 서버 없음 | 완전 클라이언트 사이드 동작. 다른 기기 간 동기화 불가 |
| 🗑️ 데이터 소멸 | 브라우저 캐시/데이터 삭제 시 저장 목록 전체 소멸 |
| 📏 길이 제한 | 암호 생성 길이는 4~50자로 제한 (입력값 범위 강제 적용) |
| ⌨️ Enter 지원 | 마스터 비밀번호 입력란에서 Enter 키로 확인 가능 |
| 📱 반응형 | 모바일/데스크탑 모두 지원 (Tailwind 반응형 클래스 적용) |
